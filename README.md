# Home Care Agency — Simple Flask Site

This is a minimal Flask-based website for a home care agency. Replace the placeholders in the templates with your real address, phone, email and services.

Quick start

1. Create a virtual environment (optional but recommended):

```bash
python3 -m venv venv
source venv/bin/activate
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
python app.py
```

4. Open http://localhost:5000 in your browser.

Files

- [app.py](app.py) — main Flask application
- [templates/layout.html](templates/layout.html) — base layout with placeholders
- [templates/index.html](templates/index.html) — homepage with services and about placeholders
- [templates/contact.html](templates/contact.html) — contact page with placeholders
- [static/style.css](static/style.css) — basic styles
